Automated Linkedin Poster

I generated authentication keys from the Linkedin Developer website and used them to generate an OAuth token for ServiceNow. I stored the token in a credential alias within ServiceNow so that it could be used to access the API.

I then created two actions in the ServiceNow flow designer, one to retrieve my profile data from the API and one to then send a post request to the API.Then I created a flow that checks for errors throughout the process. If an error occurs it changes the record status to "Error". If it publishes successfully the record status changes to "Published"
